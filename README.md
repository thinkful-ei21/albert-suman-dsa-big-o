Big O Notations for Curriculum Drills

- EVEN OR ODD
  - O(1), constant

- ARE YOU HERE
  - O(n^k), polynomial,
  - O(a*b), where a and b are array sizes of a and b, respectively

- DOUBLER
  - O(n), linear

- NAIVE SEARCH
  - O(n), linear

- CREATING PAIRS
  - O(n^k), polynomial

- FIBONACCI
  - O(n), linear

- EFFICIENT SEARCH
  - O(log(n)), logarithmic

- RANDOM ELEMENT
  - O(1), constant

- IS IT PRIME
  - O(n), linear
